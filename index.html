Warning!

These are the commonly used commands to work with the engine.
We ("tinking as a future developTeam-Company") have built a tool to read, assign and fire properties directly at the Javascript Objects, being this useful for testing an debugging. Scroll down to the bottom and you will find it.


--//Terminal Functions//--

info: print terminal function names, use -g for global scope

writeHistory [text to print]: print given parameter

StartEngine: Starts the 3d engine objects and main camera.

assign [object] [value]: assign object values, use -g to target global scope.

--//Run only at Engine//--

setFPV: Starts the camera at FirstPersonView.

generateWorld: render some randomly positionated objects.

acelerate: start object motion.

stopMotion: stops object motion.
